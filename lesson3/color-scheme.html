Color Scheme

#4F5D73

For body background color on large desktop screens. (Media queries over 500 or so pixels.)

#282F47

For header and footer background color and maybe some text color on white.

#FFFFFF

For main background color and some text color on color. (When black doesn't provide appropriate contrast.)

#E0995E

For accents, menu hover, or possibly background color on some smaller blocks.

#9A5C38

For background color on menu and certain accented blocks.

#B7B5B3

When design calls for a neutral color.
